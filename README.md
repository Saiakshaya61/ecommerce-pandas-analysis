E-commerce Sales Analysis using Pandas

 About the Project

This project focuses on analyzing e-commerce data to understand how sales, customers, and products behave.
Using Pandas, I explored the dataset, cleaned the data, and generated insights that can help in business decision-making.

---

##  Aim

To analyze e-commerce sales data and extract meaningful insights about revenue, customer behavior, and product performance.

---

##  Objectives

* Load and understand the dataset
* Clean and prepare the data
* Create new features like Total Amount
* Analyze sales and revenue trends
* Identify top customers and products
* Study payment methods and city-wise performance

---

##  Tools & Technologies

* Python
* Pandas
* Google Colab

---

##  Dataset

The dataset contains:

* Order details
* Customer information
* Product category
* Price and quantity
* Payment method
* City and delivery details

---

##  What I Did

### 1. Data Loading

Loaded the dataset using Pandas.

### 2. Data Understanding

Used functions like `head()`, `info()`, and `describe()` to explore the data.

### 3. Data Cleaning

Checked for missing values and handled them.

### 4. Feature Engineering

Created a new column:

* **Total_Amount = Price × Quantity**

### 5. Data Analysis

Performed multiple analyses:

* Total revenue
* Top-selling products
* Customer spending behavior
* Revenue by city
* Payment method usage
* Delivery and rating insights

---

 Key Insights
Electronics category generated highest revenue
Top customers contributed majority of sales
UPI is the most used payment method
Certain cities show higher purchasing activity
